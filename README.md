# Camera Scan — Premium Mobile Digital Portfolio

A single-page mobile-first digital business card for **Camera Scan**, serving Kottayam since 1993.

## Tech Stack

- **Next.js 15** with App Router
- **TypeScript**
- **TailwindCSS** with custom design system
- **Framer Motion** for animations
- **Lucide Icons**
- **PWA Ready** (manifest + service worker)

## Features

- Single page mobile digital portfolio
- Splash screen with logo animation
- Sticky floating header with slide-in menu
- Hero with animated background and floating camera icons
- Glassmorphism business info card with copy-to-clipboard
- 9 quick action buttons (Call, WhatsApp, Email, Instagram, Website, Maps, Share, Download, Save)
- vCard download (CameraScan.vcf)
- navigator.contacts API support with vCard fallback
- navigator.share() with clipboard fallback
- About, Services grid, Authorized Brands marquee
- Why Choose Us with animated statistics counters
- Masonry gallery with fullscreen lightbox
- Embedded Google Maps
- Business hours
- Auto-scrolling customer reviews
- Premium contact card
- QR code to this page
- Floating WhatsApp FAB
- Sticky bottom action bar (5 icons, glass effect)
- Toast notifications
- Dark mode support
- SEO: Open Graph, Twitter Cards, Schema.org LocalBusiness
- Micro animations: fade, scale, slide, float, ripple, card lift

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
